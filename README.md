## 🚀 Implementação Técnica (SEO & GTM)
O projeto utiliza o **Google Tag Manager (GTM-5QTLB9JC)** para a injeção dinâmica de esquemas de dados (JSON-LD). Esta arquitetura garante que o escritório seja reconhecido como uma entidade verificada e geograficamente localizada pelos motores de busca.

### Pilares do Projeto:
* **Local SEO (LegalService):** Implementação de coordenadas geográficas precisas e informações de contato validadas.
* **E-E-A-T (AboutPage & Person):** Estruturação da trajetória profissional do Dr. Julio para aumentar a confiabilidade algorítmica.
* **Arquitetura Versionada:** Gestão de scripts via GitHub para garantir portabilidade e histórico de alterações.
